# Landing Page

## Setup
### Backend
```bash
cd 314/backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

### Frontend
Using a split terminal
```bash
cd 314/frontend
npm install
npm run dev
```

### How to edit the code
For frontend design, most of the codes can be found in App.jsx
